CSS syntax ประกอบด้วยอะไรบ้าง ชื่อของ attribute และค่าของ attribute นั้นๆ โดยใช้ : คั่นระหว่างชื่อและค่านั้น ปิดท้ายด้วย ; เช่น color:green;
การเขียน CSS มีกี่แบบอะไรบ้าง
    มี 3 แบบคือ
  1. inline เขียนลงไปใน tag ของ element นั้นๆโดยตรง เช่น <p style="width:200px:">
  2. internal จัดกลุ่ม CSS ไว้ใน tag style เช่น 
        <style> 
        p {
            width:20px;
        }
    </style>
  3. external เขียนแยกไว้ต่างหากอีกไฟล์หนึ่ง โดยบอกตำแหน่งของไฟล์ด้วย tag link ที่ใส่ไว้ในส่วน head ของ html เช่น <link rel="stylesheet" href="">
Combinator มีกี่แบบ อะไรบ้าง
    มี 4 แบบคือ
  1. ใช้ space เปล่าๆ เช่น div p จะเป็นการเลือก p ทุกตัวที่อยู่ภายใน div นี้
  2. ใช้ > เช่น div>p จะเป็นการเลือก p ทุกตัวที่เป็นลูกคนแรกที่อยู่ภายใน div นี้
  3. ใช้ + เช่น div+p จะเป็นการเลือก p ตัวแรกที่อยู่ในเลเวลเดียวกัน ตัวถัดมาจาก div นี้
  4. ใช้ ~ เช่น div~p จะเป็นการเลือก p ทุกตัวที่อยู่ในเลเวลเดียวกัน ที่ถัดมาจาก div นี้
